# Lyra Secretary v1.1

Lyra Secretary is the first organ of a personal cognitive operating system where the adaptive scheduler functions as the nervous system.

## Project Structure
- `backend/` - FastAPI backend application handling state, rules, and database
- `openclaw/` - Cognitive layer orchestrating interactions

## Setup
1. Copy `.env.example` to `.env` and fill in credentials.
2. Setup environment and databases.
