## Claims Softened

- Baseet is framed as the academic structure provider for the pilot, not as
  LyraOS itself.
- Provider data is framed as context for planning and recovery, not proof of
  learning, completion, attention, or understanding.
- Brain dump is framed as optional fallback input for provider-incomplete or
  non-Baseet work, not the core Baseet value.
- Adaptive scheduling, AI synthesis, provider-progress candidates, and
  session-continuity estimates remain future-gated.
- University value is framed as aggregate pilot learning about workload shape
  and support design.

## V4 Presentation Choices

- Success metrics are survey-first with supporting dashboard metrics.
