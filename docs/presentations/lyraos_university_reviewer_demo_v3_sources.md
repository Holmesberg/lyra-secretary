## Claims Softened

- Baseet is framed as the academic structure provider for this pilot, not as
  LyraOS itself.
- Provider data is framed as context for planning and recovery, not proof of
  learning, completion, attention, or understanding.
- Adaptive scheduling, AI synthesis, provider-progress candidates, and
  session-continuity estimates remain future-gated.
- University value is framed as aggregate pilot learning, not individual-level
  student reporting.
- LyraSim appears only in backup because it answers technical confidence
  questions without interrupting the main value story.

## V3 Presentation Choices

- Main deck: 12 slides.
- Backup appendix: 8 slides.
- Main-slide language avoids raising objections before the value is clear.
- Success metrics are survey-first, supported by dashboard usage metrics.
- The Insights screenshot is deliberately enlarged and placed on the right for
  readability.
