# User Burden Surface

## Definition

The set of required user actions, prompts, fields, and decisions needed to use LyraOS.

## Why It Matters

Research value cannot be purchased by casually increasing friction.

## Where It Appears

- task creation
- timer start/stop
- readiness and reflection
- repair prompts

## Failure Mode

The product becomes a research instrument that users stop using.
