# Operator Runtime Map

Operator tooling helps build, inspect, and reason about LyraOS. It is not product research data by default.

## Surfaces

- JARVIS in-app assistant
- OpenClaw multi-agent runtime
- Telegram operator notifications
- Obsidian vault synthesis

## Boundary

Operator output may help understanding and implementation. It must not silently create Lyra schema, behavioral labels, clean-data profiles, or product research claims.
